This Elementor plugin allows you to display and format a code container that includes for the viewer an easy to use copy to clipboard functionality.

== Description ==

ingenidev, Code Widget for Elementor adds a Widget to your Elementor menu providing a drag and drop, easy to use code snippet that automatically formats the contents as code and allows the page viewer to easily copy the code contents to his clipboard for ease of use.


= Privacy notices =

 This plugin, in itself, does not:

* track users
* write any user personal data to the database
* send any data to external servers
* make use of cookies
